# Macro for private QWERTY Discord server
`F3` To Activate Macro, <br />
`F8` To Stop Macro <br />
The Macro Does the following <br />
- Types `!work`, <br />
- Hits *enter*, <br />
- Types `!dep all`, <br />
- Hits *Enter* again <br />
- Waits **Either 5, 6, or 7 minutes** <br />
- Repeats From Typing `!work` <br />
